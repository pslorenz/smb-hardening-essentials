Findings

Failures are ordered by severity. Critical and high-severity items close real attack paths and should be remediated first.

Critical SMB-IAM-003 Identity

Legacy authentication is blocked tenant-wide

No Conditional Access policy blocking legacy auth was detected. Security Defaults is disabled.

Remediation

Legacy authentication bypasses MFA. Either enable Security Defaults (Entra ID → Properties → Manage Security Defaults) or create a Conditional Access policy targeting 'Other clients' and 'Exchange ActiveSync' with Block access. Verify no critical line-of-business integrations depend on legacy auth before enabling.

Critical SMB-OFF-001 Office

Block macros from internet (Word) — Mark of the Web enforcement

Registry value 'blockcontentexecutionfrominternet' at HKCU:\SOFTWARE\Policies\Microsoft\office\16.0\word\security is 0; expected 1.

Remediation

This is the single highest-ROI Office setting. Deploy via Intune Administrative Template or Group Policy. User > Policies > Administrative Templates > Microsoft Word > Word Options > Security > Trust Center > Block macros from running in Office files from the Internet.

High SMB-IAM-004 Identity

Global Administrator count is between 2 and 4

Seven accounts currently hold the Global Administrator role.

Remediation

Reduce Global Admin count to 2-4. Assign lower-privileged roles (Exchange Administrator, User Administrator, Security Administrator) for role-specific duties. Retain one designated break-glass account.

High SMB-EXO-003 ExchangeOnline

Automatic forwarding to external domains is blocked

Current value: Automatic. Default outbound spam policy allows auto-forward to external recipients.

Remediation

Classic post-compromise exfiltration path. Set-HostedOutboundSpamFilterPolicy -Identity Default -AutoForwardingMode Off. Verify no legitimate forwarding rules depend on this before enforcing.

High SMB-SPO-001 SharePoint

External sharing capped at 'New and existing guests'

Tenant external sharing is set to 'Anyone', enabling anonymous link creation.

Remediation

SharePoint admin center → Policies → Sharing → set external sharing to 'New and existing guests'. Existing 'Anyone' links will continue to work until expiration; new 'Anyone' links cannot be created.

High SMB-CRED-001 Credential

LSA protection (RunAsPPL) enabled

Registry value 'RunAsPPL' at HKLM:\SYSTEM\CurrentControlSet\Control\Lsa is 0; expected 1.

Remediation

Blocks LSASS memory access by non-PPL tools, a core mitigation against credential theft. Deploy via Intune or GPO. Some legacy security agents may conflict — validate with endpoint protection vendor first.

Medium SMB-EXO-007 ExchangeOnline

DMARC policy is at least p=quarantine with reporting

_dmarc.acmecorp.com record is present with p=none. No rua= reporting tag.

Remediation

Publish TXT at _dmarc.acmecorp.com: v=DMARC1; p=quarantine; rua=mailto:dmarc@acmecorp.com; fo=1. Start with p=quarantine and weekly review of aggregate reports; graduate to p=reject once legitimate senders are clean.

Medium SMB-DEF-005 Defender

PUA protection block mode

Registry value 'PUAProtection' at HKLM:\SOFTWARE\Policies\Microsoft\Windows Defender is 0; expected 1.

Remediation

Catches bundled adware and legitimately unwanted applications. Low false-positive rate. Set-MpPreference -PUAProtection Enabled.

Medium SMB-NET-004 Network

LLMNR disabled

Registry value 'EnableMulticast' at HKLM:\SOFTWARE\Policies\Microsoft\Windows NT\DNSClient is 1; expected 0.

Remediation

LLMNR is the classic NTLM relay vector exploited by Responder-class tools. Disable via GPO: Computer Configuration → Administrative Templates → Network → DNS Client → Turn off multicast name resolution → Enabled.

Low SMB-AUD-002 Audit

Audit log retention at tenant maximum

Default 90-day retention in effect. E3 license supports 180 days; upgrade available at no additional cost.

Remediation

Extended retention improves incident response capability. Compliance & Security → Audit → Retention policies → create policy at 180 days for all workloads.

What this report is — and isn't

This assessment measures your Microsoft 365 and Windows endpoint configuration against a curated baseline sized for small and mid-sized organizations. It does not measure attacker behavior, employee behavior, or the effectiveness of licensed security products at runtime. A passing score reflects good hygiene, not invulnerability. A failing score reflects improvement opportunities, not imminent breach.

Controls deferred to the Tier 2 baseline (available separately) address more advanced threats that require dedicated operational capacity to maintain. Organizations with compliance obligations, in-house IT staff, or higher risk tolerance should consider the enhanced baseline.
